Desafio

Hoje você irá acessar páginas html com uma gama de tags com diversas funcionalidades, como:

  • As tags H1 a H6 são utilizadas para definir títulos e subtítulos em um documento HTML.
  • As tags P, Span e Div são usadas para organizar e formatar conteúdo em um documento HTML.
  • As tags Ol, ul e li são usadas para criar listas em um documento HTML.
  • As tags Strong, em, del e ins são usadas para enfatizar ou modificar o texto em um documento HTML.
  • As tags Img, audio e video são usadas para inserir conteúdo multimídia em um documento HTML.
  • As tags Form, input e button são usadas para criar formulários em um documento HTML.
  • A tag a (ancora) é usada para criar links em um documento HTML.
